# Admin Dashboard

A modern, responsive admin dashboard built using HTML, CSS Grid, and Flexbox. This project focuses on layout structure, clean UI design, and reusable component-based sections.

## 📌 Features

- Responsive dashboard layout using CSS Grid
- Sidebar navigation with SVG icons
- Header with search bar, user info, and action buttons
- Project cards with descriptions and actions
- Announcements and trending sections
- Glassmorphism-inspired UI styling
- Clean, modern color palette

---

## 🛠️ Built With

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Material Design Icons (SVG)

---

## 🧠 What I Learned

- Structuring complex layouts using nested CSS Grid systems
- Understanding parent-child relationships in layout design
- Debugging layout issues caused by fixed heights
- Organizing UI into reusable components (cards, panels, navigation)
- Applying modern UI techniques like glassmorphism
